CSS 속성(property) float 은 한 요소(element)가 보통 흐름(normal flow)으로부터 빠져 텍스트 및 인라인(inline) 요소가 그 주위를 감싸는 자기 컨테이너의 좌우측을 따라 배치되어야 함을 지정합니다.

        시도해보기

        부동**(floating) 요소** 는 float 의 계산값(computed value)이 none이 아닌 요소입니다.

        초기값	none
        적용대상	all elements, but has no effect if the value of display (en-US) is none.
        상속	no
        계산 값	as specified
        Animation type	discrete
        float 은 블록 레이아웃의 사용을 뜻하기 때문에, 일부 경우에 display (en-US) 값의 계산값을 수정합니다:

        지정값(Specified value)	계산값
        inline	block
        inline-block	block
        inline-table	table
        table-row	block
        table-row-group	block
        table-column	block
        table-column-group	block
        table-cell	block
        table-caption	block
        table-header-group	block
        table-footer-group	block
        flex	flex, 다만 float 은 이러한 요소에 효과가 없음
        inline-flex	inline-flex, 다만 float 은 이러한 요소에 효과가 없음
        그외	변화없음
        참고: 주의: JavaScript에서 이 속성을 element.style 객체의 멤버로서 참조한다면, 당신은 cssFloat으로 철자를 써야 합니다. 또한 Internet Explorer 8 이전 버전에서는 styleFloat으로 썼음을 주의하세요. 이는 DOM 멤버의 이름은 dash(-)로 구분된 CSS 이름의 카멜케이스(camel-case) 명이다는 규칙의 예외(이고 "class"는 "className"으로
